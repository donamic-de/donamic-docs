# Konfiguration

Dieses Kapitel richtet sich an **Administratoren**.

## Als Startseite festlegen

Sie können Dashboards Pro anstelle der klassischen i-doit-Startseite anzeigen lassen.
Die Einstellung nehmen Sie in Ihren persönlichen **Benutzer-Einstellungen** vor; sie
gilt jeweils nur für Ihren eigenen Zugang.

Ist die Einstellung aktiv und haben Sie ein Standard-Dashboard markiert, erscheint
dieses Dashboard direkt beim Öffnen der i-doit-Startseite. Der gewohnte linke
Seitenbaum bleibt dabei erhalten — nur der Startseiten-Bereich wird durch Ihr
Dashboard ersetzt.

## Command Palette (Strg+K)

Die **Command Palette** steht auf allen Seiten des Add-ons zur Verfügung und
ermöglicht eine Schnellsuche über Objekte, Dashboards und Menüeinträge. Sie ist
automatisch aktiv — es ist keine Konfiguration nötig. Der Shortcut **Strg+K**
(macOS: **Cmd+K**) öffnet die Suche.

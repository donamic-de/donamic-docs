# Command Palette nutzen

- **Strg+K** (bzw. **Cmd+K**) öffnet die Palette.
- Tippen Sie den Suchbegriff — die Palette findet Dashboards, Objekte und
  Menüeinträge.
- Die Pfeiltasten navigieren, **Enter** öffnet den Treffer.
- **Esc** schließt die Palette.

# Berechtigungen

Der Zugriff auf donamic Inheritance lässt sich über das i-doit-Rollensystem
steuern. Es gibt zwei Rechtebereiche:

| Bereich | Mögliche Rechte |
|---|---|
| **Konfiguration** | Ansehen, Erstellen, Bearbeiten, Löschen von Vererbungsregeln |
| **Log** | Ansehen des Protokolls |

## Standard-Rechte

Bei der Installation werden sinnvolle Voreinstellungen gesetzt:

- **Administratoren** dürfen alles.
- **Editoren/Autoren** dürfen Regeln bearbeiten und das Log einsehen.
- **Leser** dürfen Regeln und Log nur ansehen.

Wer über kein Recht verfügt, erhält beim Öffnen den Hinweis „Sie verfügen nicht
über die notwendigen Rechte …". Nur Benutzer mit Bearbeiten-Recht sehen eine
Regel im Bearbeitungsmodus; alle anderen sehen sie ausschließlich lesend.

Die Rechte passen Sie wie gewohnt unter **Verwaltung → Rechte** je Rolle an.

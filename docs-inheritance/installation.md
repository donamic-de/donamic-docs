# Installation

Dieses Kapitel richtet sich an **Administratoren**.

## Systemvoraussetzungen

| Voraussetzung | Wert |
|---|---|
| i-doit Edition | **Pro** |
| Browser | Aktuelle Version von Firefox, Chrome, Edge oder Safari |
| Lizenz | Gültige Lizenz für `donamic_inheritance` (siehe unten) |

## Add-on installieren

1. Melden Sie sich in i-doit als Administrator an.
2. Öffnen Sie **Verwaltung → Module → Add-on Uploader**.
3. Laden Sie das ausgelieferte ZIP-Paket hoch.
4. Öffnen Sie **Verwaltung → Module-Verwaltung**, suchen Sie den Eintrag
   **Vererbung** und klicken Sie auf **Installieren**.

Nach der Installation finden Sie den Menüpunkt **Vererbung** im Hauptmenü.

## Lizenzierung

donamic Inheritance ist lizenzpflichtig. Ohne gültige Lizenz erscheint beim
Öffnen des Add-ons ein Hinweis mit einem Link zur Lizenzverwaltung.

- Ist das Add-on bereits über Ihre i-doit-Lizenz abgedeckt, müssen Sie nichts
  weiter tun.
- Andernfalls hinterlegen Sie Ihren donamic-Lizenzschlüssel unter
  **Verwaltung → Add-ons → donamic Lizenzen**.

Bei Fragen zur Lizenz wenden Sie sich an
[support@donamic.de](mailto:support@donamic.de).

# Dashboards teilen

## Freigaben anlegen

1. Öffnen Sie das Dashboard und klicken Sie auf das **Teilen**-Symbol (Personen-Icon).
2. Wählen Sie einen der folgenden Empfänger:
   - **Person**: Einzelner i-doit-Nutzer.
   - **Gruppe**: Alle Mitglieder einer i-doit-Personen-Gruppe erhalten Zugriff.
3. Wählen Sie das Recht:
   - **Ansehen** (`view`): Nutzer kann das Dashboard öffnen, aber nicht ändern.
   - **Bearbeiten** (`edit`): Nutzer kann Widgets hinzufügen, umkonfigurieren und das
     Layout ändern. **Löschen ist weiterhin ausgeschlossen.**
4. Speichern.

## Freigaben entziehen

Die Freigaben-Liste im Teilen-Dialog zeigt jeden Eintrag mit einem Entfernen-Symbol.
Entzogene Rechte wirken sofort.

## Öffentlicher Link

- Erzeugen Sie im Teilen-Dialog einen **öffentlichen Link**. Das Dashboard wird dann
  ohne Login erreichbar.
- Optional mit **Ablaufdatum** — nach Ablauf schlägt der Zugriff fehl.
- Jeder Aufruf erhöht den Zugriffszähler (zur Überwachung).
- Links lassen sich jederzeit widerrufen.

:::warning Sicherheitshinweis
Öffentliche Links umgehen das Berechtigungssystem. Prüfen Sie vor der Erstellung,
welche Daten auf dem Dashboard sichtbar sind.
:::

# Dashboards verwalten

## Dashboard anlegen

1. Öffnen Sie **Meine Dashboards** im Hauptmenü.
2. Klicken Sie **+ Neues Dashboard**.
3. Vergeben Sie einen Titel und optional eine Beschreibung.
4. Das leere Dashboard öffnet sich im Bearbeitungsmodus.

## Dashboard bearbeiten

- Klicken Sie oben rechts auf **Bearbeiten**, um Widgets hinzuzufügen, zu verschieben
  oder zu entfernen.
- Widgets lassen sich per Drag-and-Drop auf dem Raster anordnen; das Layout wird
  automatisch gespeichert.
- Über das Zahnrad-Symbol eines Widgets öffnen Sie dessen Konfiguration.

## Dashboard löschen

- Nur der Besitzer (und Admins) kann ein Dashboard löschen.
- Menü **⋮** → **Löschen**.

## Als Standard-Dashboard markieren

- Das als Standard markierte Dashboard wird geöffnet, wenn Sie „Dashboards Pro als
  Startseite" aktiviert haben oder die Liste über den Hauptmenü-Eintrag aufrufen.
- Menü **⋮** → **Als Standard festlegen** (Stern-Symbol).

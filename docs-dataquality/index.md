# Data Quality

**donamic Data Quality** liefert regelbasiertes Monitoring der Datenqualität in der
i-doit-CMDB. Administratoren definieren in einem grafischen Regel-Editor Pflichtfelder,
Mindestmengen und Wertprüfungen pro Objekttyp; das Add-on wertet diese Regeln gegen den
Datenbestand aus, berechnet einen Compliance-Score und stellt zahlreiche Auswertungs-
und Verwaltungs-Sichten bereit — von der KPI-Übersicht über veraltete Objekte und
isolierte Beziehungen bis hin zu IP-Konflikten, Vertragsabläufen, Duplikat-Erkennung
und Lizenz-Allokation.

Jedes Objekt erhält über eine globale Kategorie (**Datenqualität**) eine eigene
Score-Anzeige direkt in der CMDB. Trends werden täglich in einem Snapshot festgehalten,
sodass sich der Verlauf über Wochen und Monate (Zeitraum-Auswahl 7 / 30 / 90 / 365 Tage)
auswerten lässt.

## Zielgruppe

Dieses Handbuch richtet sich an i-doit-Administratoren (Installation, Lizenzierung,
Berechtigungen, Konfiguration) und an Endanwender (Bedienung der Auswertungs-Sichten).

# Mein Arbeitsbereich

Persönlicher Filter über die wichtigsten Sichten, eingeschränkt auf die für den
eingeloggten Nutzer relevanten Objekte. Die Zuordnung erfolgt über die
**Kontakt-Rollen** am Objekt (z. B. „Administrator", „Eigentümer").

Die **Workspace-Einstellungen** (nur mit Recht „Administrieren") legen fest, welche
Kontakt-Rollen den persönlichen Arbeitsbereich speisen. Die Konfiguration listet **alle
in der Installation vorhandenen Kontakt-Rollen dynamisch** auf — inklusive selbst
angelegter, benutzerdefinierter Rollen.

# Berechtigungen

donamic Disposal nutzt das i-doit-Rollensystem und erlaubt eine besonders feine
Steuerung: Sie können festlegen, **welche einzelnen Schritte** eine Rolle abschließen
darf. So gibt z. B. nur die Leitung Schritt 2 frei, nur die Finanzabteilung
Schritt 4.

| Bereich | Mögliche Rechte |
|---|---|
| **Konfiguration** | Ansehen, Bearbeiten der Einstellungsseite |
| **Entsorgung** | Ansehen, Löschen von Entsorgungsvorgängen |
| **Entsorgungsschritt** | Ausführen — pro Rolle wird festgelegt, welche der Schritte 1–5 sie abschließen darf |

Fehlt ein benötigtes Recht, erscheint ein entsprechender Hinweis, und die Aktion ist
nicht möglich. Die Rechte passen Sie wie gewohnt unter **Verwaltung → Rechte** je
Rolle an.

:::tip Zuständigkeiten sauberer abbilden
Nutzen Sie das **Entsorgungsschritt**-Recht, um den Freigabeprozess an Ihre
Organisation anzupassen: Jede Abteilung erhält genau das Recht für „ihren" Schritt,
sodass niemand versehentlich eine fremde Freigabe erteilen kann.
:::

# Berechtigungen

Der Zugriff auf donamic Design wird über das i-doit-Rollensystem gesteuert. Es gibt
einen Rechtebereich **Konfiguration** mit folgenden Stufen:

| Recht | Erlaubt |
|---|---|
| **Ansehen** | Designs und Einstellungen einsehen |
| **Anlegen** | Neue Designs erstellen |
| **Bearbeiten** | Bestehende Designs ändern |
| **Löschen** | Designs entfernen |
| **Ausführen** | Ein Design anwenden, zurücksetzen, exportieren und importieren |

Besonders das Recht **Ausführen** ist entscheidend: Nur damit lässt sich ein Design
tatsächlich für alle Benutzer aktivieren oder zurücksetzen. So können Sie das
Erstellen von Designs breiter erlauben, das Aktivieren aber einem kleinen Kreis
vorbehalten.

Die Rechte passen Sie wie gewohnt unter **Verwaltung → Rechte** je Rolle an.
